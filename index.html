What is Craft?

مش كورس… ورشة صنعة حقيقية

تطوير ويب حقيقي

How It Works

كلهم مطوّرين — كل واحد بيبني جزء

مفيش أدوار مختلفة — كل طفل في الفريق مطوّر. بنقسّم الموقع لأجزاء، كل واحد بيبني جزئه، وفي النهاية بيربطوهم مع بعض.

الكل بيكتب كود

مفيش حد بس بيشوف أو بيدير — كل طفل بيكتب HTML و CSS بإيده ويشوف النتيجة أمامه على الشاشة.

كل واحد ليه جزء

بيتقسّم الموقع لأجزاء — هيدر، محتوى، فورم — وكل طفل مسؤول عن جزء كامل.

بيربطوهم مع بعض

في الأسبوع التالت بيلمّوا كل الأجزاء في ملف واحد — ويشوفوا موقعهم كامل بيشتغل.
